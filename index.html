How It Works

PlayCord creates an interactive Thread for each active match to prevent chat clutter while you play. Players simply use Discord's intuitive Slash Commands or UI Buttons (depending on the game) to interact, take their turns, and strategize.

Games track your matchmaking rating using the TrueSkill rating system, maintaining fair competition across servers! For more information on matches, you can even check out replay logs via /playcord replay or review histories via /playcord history.

Features

Vast Game Library

Play classic games right in Discord with dedicated UI, from logic games to card mechanics correctly integrated.

Global Leaderboards

Every match calculates your TrueSkill rating, ranking you globally or per-server with built-in matchmaking.

Match Replays

A replay event log is recorded for most games with stochastic elements and logic, enabling playback after via slash commands.

Developer Friendly

PlayCord is built on an extensible API, meaning custom games and frameworks can easily be written. A full web API documentation is available.
