Mastermind

Objective

Crack the hidden code in as few guesses as possible.

How Feedback Works

  • Exact match: right symbol in the right position.
  • Partial match: right symbol in the wrong position.
  • No match means that symbol is not in the code or already fully counted.

Turn Flow

  1. Submit a full-length guess.
  2. Review exact and partial feedback.
  3. Refine next guess based on remaining possibilities.

Win and Loss

  • Win by guessing the full code exactly.
  • Lose if guess limit is reached without cracking the code.

Strategy Tips

  • Start with a guess that tests repeated and distinct symbols.
  • Track eliminated symbols carefully.
  • Change one variable at a time once close to solution.
